¶加法-2 😂

-$2^{63}$≤ a、b < $2^{63}$ , 0 < mod < $2^{63}$ ，由于long long 最大为 $2^{63}$-1 , a+b 会爆long long ,在判断过程中要注意是否超出范围.

该题应该想到判断 a , b 的正负性,分为一下几种情况

  1. a < 0 && b < 0
  • 据题意，当 a + b 的结果是一个负数，那么我们直接输出0.
  1. a < 0 || b < 0
  • 此情况下 a + b $\in$ ( 0 , $2^{63}$-1 ] 不会爆 long long ,所以直接相加即可.
  1. a > 0 && b > 0

  • 取模性质 : ( a + b ) % mod = ( a % mod + b % mod ) % mod

  • 但这个在此题无法适用 , 由于 0 < mod < $2^{63}$ ,两个mod数相加，仍会爆 long long , 但是我们只要判断 两个数相加是否大于 long long max ，再分两种情况讨论

    • a % mod + b % mod <= long long max 正常应该直接是这样判断，但上面分析会爆值 ，因此做一个简单的移位

      a % mod <= long long max - b % mod

      在这种情况下直接输出 a + b 的值

    • a % mod + b % mod > long long max 相反，这就是另一种情况，此时该如何求值呢

      ​ ( a + b ) % mod = a % mod - mod + b % mod ( mod < a + b )

      左式为要求的值，在 ( mod < a + b ) 时右式等价于左式，在 a + b 中间减去一个mod的值, 不改变原式,同时也不会爆 long long ，这样问题就解决了

      接下来我们要证明这个等式

      • a % mod < mod , b % mod < mod

      • mod < ( a % mod + b % mod ) < 2 * mod

      • ( a + b ) / mod = 1 … ( a + b - mod )
      • a % mod + b % mod - mod = ( a + b ) % mod
