学习过程中的一些总结,不吝赐教.

此章节为动态规划的引入.

¶数字三角形

DP的入门题

由于在转移方程中只涉及到了 i - 1行和 i 行,思考是否可以将其压缩为一维数组

因为这题要用到i行中的数据,所以不能像01背包一样直接一维优化,可以使用一边读入一边转移的方式来完成

这题还可以倒着往上推,当出现负数时，以下推导仍然适用.

01背包模板题，不过多赘述

¶挖地雷

¶滑雪

记忆化搜索

¶最大子段和

要选出其中连续且非空的一段使得这段和最大

要么从第i个数为起点开始选，要么第i个数为终点

状态表示: f[i]以第i个数字为终点且一定包含第i个数字的连续最大子段和

转移方程 : f[i] = max(a[i] ,f[i-1]+a[i])

¶五倍经验日

01背包的变式, f[i]表示用i瓶药可以获得的最多的经验

  • 当 v[i] <= j <=x 时,可以选择不被打败和被打败，当不被打败时根据贪心直接不用药
    • f[j] = max(f[j] + l[i], f[j - v[i]] + w[i])
  • 当 0 <= j < v[i] 时,只能被打败
    • f[j] = f[j] + l[i]

¶过河卒

状态表示: g[i] [j] 表示从(1,1)走到(i,j)总共的路径数

转移方程: g[i] [j] = g[i - 1] [j] + g[i] [j - 1];

只要把马能走到的地方当成障碍就可以,其位置路径直接变为0

¶装箱问题

要求出最小的剩余空间，也就是要求出最大的可装重量.

与01背包相似，当我们把一个物体的重量看作它的价值，那么完全可以转换成一个01背包问题.

01背包中 f[j] 表示当背包容量为j时,能够装的最大的物品价值,

此体重，f[i]表示当背包承重为j时，能够装的最大的物品重量.

¶疯狂的采药

完全背包问题,不过多赘述.

¶小A点菜

此题为01背包的变式.

¶摆花

01背包变式

一维优化

注意本题一维优化在考虑j要倒序的同时,不能直接把第一维删掉,

因为当k==0的时候 f[j] 就是继承了上一行的数据 ,所以f[j] = f[j] +f[j-0]导致错误,所以此题一维优化k!=0

¶线段

由于要把一条线段全部走完，所以一条线段的终点要么就是左端点，要么就是右端点

那么这条线段的起点可以是上一条线段的左端点，也可以是上一条线段的右端点

要走到线段的左端点一定是从线段的右端点过来，也就是从上一条线段的左端点或右端点出发到本线段的右端点，在到终点左端点

要走到线段的右端点同理

¶金明的预算方案

01背包的变式

我们在做01背包的时候，每次有两种选择，一种不不选择该物品，一种是选择该物品

此题同理，每次有5种选择

  • 什么都不选
  • 选对应的主件
  • 选对应的主件和对应主件的第一件附件
  • 选对应的主件和对应主键的第二件附件
  • 选对应的主件和对应主件的第一件附件和对应主键的第二件附件

¶考前临时抱佛脚

一个费用与价值相同的背包问题

发现当左右两脑需要处理的题目花费时间越接近，最终花费时间越短

那么我们把题目分别放进左右脑，尽可能保持55分

背包结束后 f[HALF]总是小于等于S-f[HALF],应累加较大值即后式
